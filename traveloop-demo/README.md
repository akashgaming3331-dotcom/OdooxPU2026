# 💻 Traveloop Web Demo

A responsive and interactive web demonstration of the Traveloop platform, built with [Next.js](https://nextjs.org/) and [Framer Motion](https://www.framer.com/motion/).

## ✨ Highlights

-   **Modern UI/UX**: Built with Tailwind CSS 4 and Lucide React icons.
-   **Smooth Animations**: Fluid transitions and interactive elements using Framer Motion.
-   **Dashboard View**: Overview of active and upcoming trips.
-   **Responsive Design**: Optimized for desktop, tablet, and mobile browsers.

## 🚀 Getting Started

### Installation
```bash
npm install
```

### Running Locally
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the application.

## 🛠️ Tech Stack
-   **Framework**: Next.js 16 (App Router)
-   **Styling**: Tailwind CSS 4
-   **Icons**: Lucide React
-   **Animations**: Framer Motion
-   **Language**: TypeScript
