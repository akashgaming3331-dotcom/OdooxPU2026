# 📱 Traveloop Mobile App

The native companion for the Traveloop ecosystem, built with [Expo](https://expo.dev/) and [React Native](https://reactnative.dev/).

## 🌟 Features

-   **Native Experience**: High-performance mobile UI using React Native Reanimated.
-   **File-based Routing**: Built with Expo Router for intuitive navigation.
-   **Trip Companion**:
    -   View itineraries on the go.
    -   Manage packing checklists with haptic feedback.
    -   Quickly add trip notes and photos.
    -   Real-time expense tracking.
-   **Authentication**: Secure mobile login and persistent sessions.

## 🚀 Getting Started

### Prerequisites
-   [Expo Go](https://expo.dev/go) app on your device or an emulator (Android/iOS).

### Installation
```bash
npm install
```

### Running the App
```bash
npx expo start
```
Scan the QR code with your Expo Go app or press `a` for Android or `i` for iOS.

## 🛠️ Tech Stack
-   **Framework**: Expo
-   **Navigation**: Expo Router (Stack/Tabs)
-   **Icons**: Lucide React Native
-   **Animations**: React Native Reanimated
-   **Storage**: Expo Secure Store & Async Storage
