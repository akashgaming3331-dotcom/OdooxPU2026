## Key UI Elements from Wireframe

### Login Screen (Screen 1)
- Logo/Photo top center
- Login Button
- Registration link

### Registration Screen (Screen 2)
- First Name, Last Name fields
- Email Address, Phone Number fields
- City, Country fields
- Additional Information textarea
- Register Users button

### My Trips / Home (Screen 3)
- Traveloop header
- Search bar
- Group by / Filter / Sort by controls
- Plan a Trip CTA
- Previous Trips section
- Preplanned Trips section
- Trip cards with "View" buttons

### Create Trip (Screen 4)
- Select a Place
- Start Date, End Date
- Banner Image upload
- Suggestions for Places/Activities

### Build Itinerary (Screen 5)
- Traveloop header
- Section 1, 2, 3 with Date Ranges + Budget
- Budget per section
- "Add another Section" button

### Itinerary View + Budget (Screen 9)
- Day 1, Day 2 view
- Physical Activity | Expense columns
- Search bar for itinerary
- Group by / Filter / Sort by

### User Profile (Screen 7)
- Image of User
- User Details with edit options
- Group by / Filter / Sort by

### Admin Panel (Screen 12)
- Manage Users section
- Popular Cities
- Popular Activities
- User Trends and Analytics

### Packing Checklist (Screen 11)
- Documents category: Passport, Flight Tickets, Travel Insurance, Hotel Confirmation
- Clothing: Casual Shirts, Trousers, Comfortable Shoes, Light Jacket
- Electronics: Phone Charger, Universal Adapter, Earphones
- 1/3, 3/4, 1/4 completion ratios
- "+ add item to checklist" button
- Share Checklist / Reset all actions

### Trip Notes/Journal (Screen 13)

- Trip: Goa & Gokarna Adventure

- Note cards with titles and content
